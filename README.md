## 📌 Todo List Web App

This is a simple and interactive **Todo List Web Application** built using **HTML, CSS, and JavaScript**. The application allows users to efficiently manage their daily tasks through a clean and user-friendly interface.

Users can add new tasks, mark them as completed, and delete tasks as needed. The app also utilizes **browser local storage** to persist data, ensuring that tasks remain saved even after refreshing or reopening the browser.

The project focuses on demonstrating core frontend development concepts such as **DOM manipulation, event handling, and state management using local storage**, while maintaining a responsive and intuitive UI.

---

### ✨ Features

* Add new tasks
* Mark tasks as completed
* Delete tasks
* Persistent storage using LocalStorage
* Simple and responsive design

---

### 🛠️ Technologies Used

* HTML
* CSS
* JavaScript (Vanilla JS)
